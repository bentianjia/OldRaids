# OldRaids

一个适用于 1.21.4 的插件，把袭击机制恢复到 1.21 之前，让旧版袭击农场能够像以前一样正常工作，并增加了一些生电向功能：

- 击杀袭击队长会再次给予“不祥之兆”效果
- 不祥之兆可以直接在村庄中触发或延长袭击
- 袭击波次的生成位置恢复到 1.21.3 之前的生成模式
- 袭击波次生成位置会通过原版劫掠兽的生成规则进行有效性验证
- 移除不祥之瓶掉落
- 袭击中掉落的不死图腾像下界合金物品一样防火

私货：想测试的话可以加入 `trap.786913.xyz`

### 配置

插件不提供配置文件或命令。看着不爽？受着（

### 构建

```bash
mvn -DskipTests package
```

编译完成的 jar 包将输出至 `target/oldraids-1.21.4-1.6.jar`。
